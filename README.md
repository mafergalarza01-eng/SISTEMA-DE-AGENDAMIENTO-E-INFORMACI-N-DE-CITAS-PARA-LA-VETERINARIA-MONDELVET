# Sistema de Agendamiento e Información de Citas - Veterinaria Mondelvet

## Descripción del Proyecto
Este proyecto consiste en el desarrollo de una plataforma orientada a optimizar la gestión de la Veterinaria Mondelvet. El sistema permite a los clientes registrar a sus mascotas, consultar información en tiempo real y agendar de manera eficiente citas médicas, estéticas o de control. 

Asimismo, ofrece al personal de la veterinaria una herramienta centralizada para administrar el calendario de atenciones, gestionar el historial de citas y enviar recordatorios informativos.
